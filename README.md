A Discord bot for roleplaying that creates character dialogue images with sprites and textboxes.

## Features

- Register characters with custom sprites and textboxes
- Create dialogue images with character sprites and text
- Simple slash commands for easy use
- Image manipulation with proper compositing

## Commands

- `/textboxregister` - Register a new character with sprite and textbox
- `/textboxedit` - Edit an existing character's images
- `/textboxremove` - Remove a character
- `/say` - Make a character speak (creates dialogue image)
- `/textboxlist` - List all registered characters
